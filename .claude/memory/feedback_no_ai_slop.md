Do NOT make websites that look AI-generated.

**Why:** User explicitly called out AI slop patterns and wants a premium, hand-crafted feel.

**How to apply:**
- NEVER use purple-to-blue gradients (the #1 AI slop pattern)
- NEVER use generic overlines like "CUSTOM SOFTWARE & AI SOLUTIONS" or "WHAT WE DO"
- NEVER use vague hero headlines like "Build the future" or "Moves Industries Forward"
- NEVER use uniform fade-up animations on every section
- NEVER use "Our Proven Process" timeline sections (generic)
- NEVER use "Technologies We Use" grid sections (filler)
- NO decorative gradients anywhere - user palette is navy + light blue + creamy gold, solid colors only
- Cards should NOT all look the same size/shape
- Use SPECIFIC copy, not aspirational vagueness
- Animations must be purposeful (state changes, attention direction) not decorative
- Hero should have dynamic visual interest (3D, product mockups, interactive elements) not gradient blobs
- Each section must earn its place - if it doesn't give users value, cut it
